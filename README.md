# Homebrew Tap for Engram

Persistent memory layer for AI agents. MCP server, REST API, TypeScript SDK, and CLI.

## Install

```bash
brew tap tstockham96/engram
brew install engram-sdk
```

## Setup

```bash
engram init
```

This auto-detects and configures Claude Code, Cursor, Windsurf, VS Code, and other MCP clients.
